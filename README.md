# 🔔 Don't Forget

> A cyberpunk-themed reminder app — because deadlines wait for no one.

![GitHub Pages](https://img.shields.io/badge/Live-GitHub%20Pages-00f5ff?style=flat-square)
![Vanilla JS](https://img.shields.io/badge/Built%20With-Vanilla%20JS-f5ff00?style=flat-square)
![PWA](https://img.shields.io/badge/PWA-Ready-ff2d78?style=flat-square)

## ✨ Features
- 🚀 Animated space background with neon stars & meteors
- 🔴 Task priorities — Urgent / Important / Normal
- 💼 Categories — Work, Study, Personal & More
- 🔔 Smart alerts with sound + browser notifications
- 🔁 Repeating reminders (every 5min, 15min, or 1hr)
- 📊 Live stats dashboard
- 📱 Installable as a Mobile App (PWA)
- 💾 Data saved locally — no account needed

## 🛠️ Tech Stack
- HTML5 + CSS3 + Vanilla JavaScript
- Canvas API (space animation)
- Web Notifications API
- LocalStorage
- Progressive Web App (PWA)
